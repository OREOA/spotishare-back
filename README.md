# Spotishare backend

## Setup

```
npm install
```

## Lint

```
npm run lint
```

## Test

```
npm run test
```

## Development

```
npm run dev
```

## Package Audit with snyk

```
npm run audit
```
## Development with Docker
1. Build image
```
make build
```
2. Start development server
```
make up
```
3. Stop development server
```
make down
```
